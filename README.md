# Fuel Efficiency Predictor

A Machine Learning web application that predicts vehicle fuel efficiency (MPG) using user inputs like engine size, horsepower, weight, etc.

---

##  Tech Stack

- Python
- Flask
- Scikit-learn
- HTML, CSS, JavaScript

---

##  Features

- Predict fuel efficiency (MPG)
- Clean and interactive UI
- Real-time prediction
- Multiple vehicle parameters supported
